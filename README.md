# Student Result Management System

A console-based Java application to manage student records.

## Features
- Add students with marks
- Auto-calculate grades (A/B/C/F)
- View all students
- Search student by ID
- Topper detection with Scholarship award (marks >= 90)
- Inheritance & Polymorphism via TopperStudent class

## Tech Stack
- Java 17
- OOPs concepts 
- ArrayList (Java Collections)
- Git & GitHub (Version Control)

## Concepts Used
- Class & Object
- Constructor
- Encapsulation
- Abstraction
- Inheritance
- Polymorphism
- ArrayList
- Exception Handling (coming soon)
- MySQL Database (coming soon)

## How to Run
1. Clone the repo
2. Open in IntelliJ IDEA
3. Run Main.java
